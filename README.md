# E-Commerce Sales Analysis Dashboard

## Tools Used
- Excel
- Power BI

## Project Overview
Analyzed 3,500 e-commerce transactions to identify sales trends, profitability, and regional performance.

## Key Metrics
- Total Sales: ₹10,667,881
- Total Profit: ₹1,844,665
- Profit Margin: 17.29%
- Total Orders: 3,500

## Key Insights
- West Region generated the highest sales.
- Electronics was the top-performing category.
- May recorded the highest sales.
- February recorded the lowest sales.

## Dashboard Features
- KPI Cards
- Sales by Region
- Sales by Category
- Profit Analysis
- Monthly Trend Analysis
- Interactive Filters
